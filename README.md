# 3D Isometric AI Developer Portfolio

An interactive, 3D isometric-inspired portfolio built for Abhimanyu Singh (Data Scientist & AI Engineer).

## Features
- **3D Floating Island**: Built entirely with HTML5 and CSS 3D transforms.
- **Interactive Parallax**: The 3D scene responds dynamically to mouse movement.
- **Micro-interactions**: Hover effects lift the 3D buildings and cast dynamic shadows.
- **Glassmorphism Detail Cards**: Beautiful, semi-transparent overlays displaying experience, projects, skills, and contact info.
- **Fully Responsive**: Automatically scales and switches to an optimized layout on mobile screens.

## File Structure
- `index.html`: Main page structure and content.
- `styles.css`: 3D engine, styling, and transitions.
- `script.js`: Mouse parallax and interactive logic.
- `static/`: Image and document assets.
